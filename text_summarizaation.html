Summarization is a very useful tool that can be utilized on various occasions or platforms such as meetings, long emails, Wikipedia pages, feedbacks from surveys etc. Motive behind trying out this project was google assitant. It gives out information of about two - three sentences when a keyword is given to it but I wanted to build a interface which would allow to interact and give more information that are summarized. The program utilizes a gensim library called TextRank algorithm which is based on PageRank algorithm which is used for ranking search results in webpages.
